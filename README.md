# DealerForge

**Train Like a Pro Dealer** — Master Roulette Payouts & Casino Math

Specialized training web application for roulette dealers (croupiers) worldwide. Inspired by SoloLearn, focused on practical casino skills.

## Features (MVP)
- Realistic AI-generated roulette payout challenges
- Accurate bet geometry (Straight, Split, Corner, Street, Six-Line)
- Timer + scoring system
- Mobile-first PWA
- Dark calm casino design

## Tech Stack
- **Next.js 15** (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Lucide React icons
- Grok AI (for tasks & images in future)
- Supabase (planned for Auth + Leaderboard)

## Quick Start

```bash
npm install
npm run dev
